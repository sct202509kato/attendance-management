# 勤怠管理システム

React + TypeScriptで構築された勤怠管理アプリケーション

## 機能

### ユーザー機能
- ✅ ログイン・ログアウト
- ✅ 新規ユーザー登録（管理者/一般ユーザー）
- ✅ 出勤・退勤打刻
- ✅ 休憩開始・終了記録
- ✅ 勤怠履歴確認
- ✅ 月次レポート確認
- ✅ 勤怠修正申請

### 管理者機能
- ✅ 全従業員の勤怠確認
- ✅ 修正申請の承認・却下
- ✅ 月次レポートのExcel出力

## 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **スタイリング**: Tailwind CSS
- **アイコン**: lucide-react
- **Excel出力**: SheetJS (xlsx)
- **データストレージ**: Claude Storage API

## セットアップ

### 1. リポジトリのクローン

```bash
git clone <repository-url>
cd attendance-management
```

### 2. 依存関係のインストール

```bash
npm install
```

### 3. 開発サーバーの起動

```bash
npm start
```

ブラウザで http://localhost:3000 が自動的に開きます。

### 4. ビルド

```bash
npm run build
```

## 使い方

### 初回セットアップ

1. **新規ユーザー登録**
   - 「新規登録」タブを選択
   - 氏名、ログインID、パスワードを入力
   - 権限（管理者/一般ユーザー）を選択
   - 「登録してログイン」をクリック

### 日常的な使用

1. **出勤打刻**: 「出勤」ボタンをクリック
2. **休憩**: 「休憩開始」→作業に戻る→「休憩終了」
3. **退勤打刻**: 「退勤」ボタンをクリック

### 修正申請（一般ユーザー）

1. 「勤怠履歴」タブで該当の記録の「修正申請」をクリック
2. 修正後の時刻と理由を入力
3. 「申請する」をクリック

### 申請の承認（管理者）

1. 「修正申請」タブを開く
2. 申請内容を確認
3. 「承認」または「却下」をクリック

### Excel出力（管理者）

1. 「月次レポート出力」タブを開く
2. 年月を選択
3. 「Excel出力」をクリック

## 注意事項

### ストレージについて

このアプリケーションはClaude Storage APIを使用しています。本番環境では以下のいずれかに移行することを推奨します：

- **Firebase**: リアルタイムデータベース + 認証
- **Supabase**: PostgreSQL + 認証
- **カスタムバックエンド**: Node.js + Express + MongoDB/PostgreSQL

### セキュリティ

- パスワードは簡易的なハッシュ化のみ実装
- 本番環境では**bcrypt**などの強力なハッシュアルゴリズムを使用してください
- HTTPS通信を必須にしてください
- セッション管理にJWTなどを使用してください
