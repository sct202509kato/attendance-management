## セットアップ手順

### 1. プロジェクト作成

```bash
npx create-react-app attendance-management --template typescript
cd attendance-management
```

### 2. 必要なパッケージのインストール

```bash
npm install lucide-react xlsx
npm install --save-dev @types/node
```

### 3. 各ファイルの配置

## 注意事項

- このプロジェクトはClaudeのストレージAPIを使用しています
- 本番環境では、バックエンドAPI（Node.js + Express、Firebase、Supabaseなど）への移行を推奨します
- パスワードのハッシュ化は簡易的な実装です。本番環境ではbcryptなどを使用してください

## 起動方法

```bash
npm start
```

ブラウザで http://localhost:3000 を開いてください。

## ビルド方法

```bash
npm run build
```

`build/` フォルダに本番用ファイルが生成されます。
