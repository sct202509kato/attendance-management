## 概要
勤怠管理アプリ（出勤・退勤・休憩管理、勤怠履歴、月次レポートのExcel出力）

## 主な機能
- 出勤 / 退勤 / 休憩管理
- 勤怠履歴一覧
- 月次レポート（Excel出力）

## 技術スタック
- React
- TypeScript
- Tailwind CSS
- Render（Static Site）

## 未実装・今後の予定

- データ永続化（Firebase / Supabase）
- ユーザー認証（管理者・一般）
- 管理者用勤怠確認画面
- 月次レポートのCSV/Google Sheets対応
